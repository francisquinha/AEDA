Detailed Description

Classe com a informacao da Biblioteca.

Notas: - nao e guardado historico dos emprestimos, mas devia ser, isto e, devia existir um outro vetor de emprestimos_antigos, com o atributo adicional data_entrega;

  • os utilizadores devem ser os mesmos que os funcionarios e o seu ID tambem, caso contrario nem todas as funcoes funcionarao corretamente.
